# Book Exchange

Sistema para intercambio de libros y material entre usuarios.

## Integrantes
- Angel David Ruiz
- Cesar Abinadi Ituarte Villalobos

## Descripción
La plataforma permitirá publicar, buscar e intercambiar libros entre estudiantes o usuarios.

# Visión del MVP - Book Exchange

## 1. El Problema (Pain Point)

Muchos estudiantes y personas tienen libros o material que ya no utilizan, mientras que otros necesitan esos recursos pero no saben dónde encontrarlos fácilmente o no pueden comprarlos.

---

## 2. Propuesta de Valor

Book Exchange permitirá a los usuarios publicar, buscar e intercambiar libros y material escolar de manera rápida y sencilla mediante una plataforma accesible para todos.

---

## 3. Alcance Crítico (MVP)

Funciones mínimas necesarias:

1. Registro e inicio de sesión de usuarios.
2. Publicación de libros o material disponible.
3. Búsqueda de libros por nombre o categoría.
4. Sistema de contacto/intercambio entre usuarios.

---

## 4. Métrica de Éxito

El MVP será exitoso si los usuarios logran publicar e intercambiar material fácilmente dentro de la plataforma y se realizan al menos varios intercambios durante las pruebas iniciales.
